# Samand Car Data Scraping

## Project Description
In this project, I collected car advertisement data from a website using web scraping.

## Problem
The main link returned only about 30 ads, which was not enough.

## Solution
To solve this, I divided the price range into 5 parts and used 5 different links to collect more data.

## Data
The dataset includes:
- price
- mileage
- color
- production year
- transmission type
- description
- title
- url

## Process
1. Collected links using Selenium
2. Extracted data from each ad
3. Cleaned and structured the data
4. Saved the final dataset

## Output
Final dataset saved as an Excel file.

## Tools
- Python
- Selenium
- Pandas
